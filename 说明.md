# JupyterLite 项目说明

## 双环境依赖安装（本地 / Pyodide）

当 notebook 需要同时在**本地 Jupyter** 和 **JupyterLite/Pyodide** 中运行时，本地已安装的包（如 `matplotlib`、`numpy`、`pint`）无需通过 `micropip` 安装，而 Pyodide 环境则需要。

使用以下写法，可在本地跳过安装、在 JupyterLite 中自动安装：

```python
try:
    import micropip
    await micropip.install(['matplotlib', 'numpy', 'pint'])
except ImportError:
    pass
```

### 说明

- **本地 Jupyter**：无 `micropip` 模块，触发 `ImportError`，执行 `pass`，不影响后续代码
- **JupyterLite / Pyodide**：存在 `micropip`，执行 `await micropip.install([...])` 安装依赖

### 备选写法（基于环境检测）

```python
import sys
if 'pyodide' in sys.modules:
    import micropip
    await micropip.install(['matplotlib', 'numpy', 'pint'])
```

使用方式：将上述单元格放在 notebook 开头，作为首个执行单元。
